Popup jQuery Plugin - v0.1

Popups when visitor scrolls downs!

Scroll Down and See the magic

jPop is Popup jQuery plugin that helps you to make awesome popup. jPop renders when a visitor scrolls down. After visitor scrolls down to 200px (As a default but you can change it). You can change its title,paragraph,Button text and class and input through initializing plugin.



How to Use

  • Html
  • javascript
<div class="popup gee-popup"> <div class="popup-inner"> <a class="close-button popup-close-button"> × </a> <div class="popup-header"> <h3 class="popup-title"></h3> </div> <p class="paragraph" ></p> <div class="user-content"></div> <button class="btn btn-info btn-block btn-lg" ></button> </div> </div>
<script type="text/javascript"> $("html").subscription({ paragraph: "ilyas karim", buttonText : "yaly yaran", buttonClass : "asd", }); </script>
